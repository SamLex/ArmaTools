Instructions

  1. Capture your animation with BIS_fnc_UnitCapture.

  2. Paste the output into the text area above.

  3. Set the allowed error threshold. Can be any number above 0 (0 yields no reduction). A larger value increases the scope for reduction but decreases animation quality.

  4. Press submit. The above text area will now contain the reduced version of the entered UnitCapture output.

  5. The reduced output can be reduced again to possibly achieve a smaller output.
